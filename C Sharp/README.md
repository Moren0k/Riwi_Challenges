# C Sharp (C#)

---

## Create folder for your project

```bash
mkdir NameOfYourApp
cd NameOfYourApp
```

## Create a new project

```bash
dotnet new console -o NameOfYourProgram

cd NameOfYourProgram
```

## Run the project

```bash
dotnet run
```
